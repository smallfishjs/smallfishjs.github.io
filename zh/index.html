smallfish

一个专注于中台系统的 React 前端解决方案

快速开始

简单

我们将开发环境中的一切都开关化配置，所有套件开箱即用；不用关心繁杂的包依赖升级问题，只需要关心 smallfish 一个版本；

规范

smallfish 将开发过程中的所有流程都制定好了规范，开发过程就像搭积木，让你的精力百分百聚焦到业务逻辑中；

统一

强约束的体系让所有开发者在 smallfish 体系下写出来的代码都规范统一，极大减轻后续维护成本；

# smallfish 是什么

smallfish 是一个专注于中台系统的 React 前端解决方案，致力于解决开发复杂中台系统的问题，让整个开发流程简单、规范、统一。

它包含了以下三部分：

  • 简单、统一、规范的前端框架 smallfish
  • 丰富的前端场景化解决方案文档
  • 配套的自动化工具

# 快速开始

$ git clone https://github.com/smallfishjs/smallfish-boilerplate-management-system.git my-app

$ cd my-app

$ npm install // or yarn

$ npm start // or yarn start

yarn(https://yarnpkg.com/) 是一个比 npm 更快更方便的包管理器
